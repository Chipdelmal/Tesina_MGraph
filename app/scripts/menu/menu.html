MGraph

MGraph is a software that allows the graphication of simulation data of artificially inserted genes in mosquito populations located in different points over a certain space and how the genes spread across these populations.

Each pie chart represents a meta-population of mosquitoes in the space. Each arc contains information of the number of mosquitoes with certain genotype existing in that node during that moment in time.

To get more information about a specific node, click on it. If you want to make them bigger, modify the radius value

Each arc contains the genotype information of the mosquitoes, it changes over the time as the mosquitoes reproduce and they interact with the other nodes

The supported alleles described in the software are:

  • H: Homing allele (Mosquitoes with the modified genotype)
  • W: Wild allele (Wild mosquitoes)
  • B: Resistant allele
  • R: Resistant allele
